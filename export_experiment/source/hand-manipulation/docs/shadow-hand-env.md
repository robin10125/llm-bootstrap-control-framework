# Shadow Hand environment (setup)

Second robot for the grab-and-lift experiment: the **Shadow Dexterous Hand** (24 joints,
20 actuators) on a translating base, mirroring the gripper task with a real dexterous hand.
This doc covers the **environment setup only** ŌĆö the harness/primitives/inject port is the
next milestone.

## What exists

- **Vendored model**: `env/models/shadow_hand/` ŌĆö DeepMind `mujoco_menagerie` Shadow Hand
  (right hand), sparse-cloned. Includes `right_hand.xml`, `assets/` (meshes), `keyframes.xml`
  (named grasp poses: `open hand`, `close hand`, `grasp sphere`, `pre grasp`, `grasp hard`, ŌĆ”),
  and `LICENSE`.
- **Composed scene**: `env/models/shadow_hand/scene_cube.xml` ŌĆö self-contained, loads from
  disk. Generated by `env/models/build_shadow_cube.py` from the stock hand:
  - forearm reoriented **palm-down** (fingers point ŌłÆz),
  - mounted on a **movable base**: `slide_x`/`slide_y`/`slide_z` joints + `base_x`/`base_y`/`base_z`
    position actuators (world forearm z Ōłł [0.40, 0.75]; home 0.55),
  - **gravity-compensated** hand (`gravcomp=1` on all `rh_*` bodies) so the base holds
    position with zero sag instead of drooping under the heavy forearm,
  - a **table** plane (z=0) and a free **5 cm cube** at the origin.

## Control / observation surface (for the next milestone)

`nq=34` (3 base + 24 hand + 7 cube freejoint), `nu=23` actuators:

- **Base (3)**: `base_x`, `base_y` Ōłł [ŌłÆ0.30, 0.30]; `base_z` Ōłł [ŌłÆ0.15, 0.20] (added to home 0.55).
- **Hand (20)**: `rh_A_WRJ2/WRJ1` (wrist), `rh_A_THJ5..THJ1` (thumb), and per finger
  `rh_A_{FF,MF,RF,LF}J4/J3/J0` (+`LFJ5`). The `J0` actuators are tendon-coupled (drive the two
  distal joints together). Ranges are in `scene_cube.xml` / printed by the characterization.
- Useful named hand poses live in `keyframes.xml` (qpos over the 24 hand joints).
- `grasp_site` sits at the palm; fingertip bodies are `rh_{ff,mf,rf,lf,th}distal`.

## Validated

Loads from disk (mesh paths resolve), settles stably (fingertips at zŌēł0.14 above the table,
cube resting at 0.025, base sag 0), steps cleanly, renders palm-down over the cube.

## Rebuild / look at it

```bash
cd harness && source .venv/bin/activate
python ../env/models/build_shadow_cube.py          # regenerate scene_cube.xml
python -m mujoco.viewer --mjcf ../env/models/shadow_hand/scene_cube.xml   # interactive 3D view
```
